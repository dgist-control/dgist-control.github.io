Overview

The PID (ProportionalГўВЂВ“IntegralГўВЂВ“Derivative) controller is the most widely used feedback control algorithm in industry and robotics. Despite its simplicity, a well-tuned PID can handle a surprisingly large class of systems.

This project covers the full lifecycle of PID design: deriving the control law from first principles, implementing it in discrete time (suitable for microcontrollers), adding robustness features (anti-windup, derivative filter), and systematically tuning gains using classical methods.

All code is self-contained and dependency-free (only NumPy + Matplotlib). Run python pid_example.py to see a step-response simulation immediately.

Control Theory

Continuous-Time PID

The standard PID control law in continuous time is:

$$u(t) = K_p\, e(t) \;+\; K_i \int_0^t e(\tau)\,d\tau \;+\; K_d\,\frac{de(t)}{dt}$$

where $e(t) = r(t) - y(t)$ is the tracking error, $r(t)$ the setpoint, and $y(t)$ the plant output.

Transfer Function Form

In the Laplace domain the PID controller is expressed as:

$$C(s) = K_p + \frac{K_i}{s} + K_d s = K_p\left(1 + \frac{1}{T_i s} + T_d s\right)$$

where $T_i = K_p/K_i$ is the integral time and $T_d = K_d/K_p$ the derivative time.

Discrete-Time Implementation

For digital controllers with sample period $T_s$, the integral is approximated by forward Euler and the derivative by a backward difference:

$$u[k] = K_p\,e[k] + K_i T_s \sum_{j=0}^{k} e[j] + \frac{K_d}{T_s}\bigl(e[k] - e[k-1]\bigr)$$

Derivative Filter

Pure differentiation amplifies noise. A first-order low-pass filter is added to the derivative term:

$$D(s) = \frac{K_d s}{\tau_f s + 1}, \quad \tau_f = \frac{T_d}{N}$$

where $N \approx 5$ГўВЂВ“$20$ is the derivative filter coefficient.

Algorithm Steps

  1. Measure the plant output $y[k]$ at each time step.
  2. Compute the error: $e[k] = r[k] - y[k]$.
  3. Accumulate the integral term: $I[k] = I[k-1] + K_i \cdot T_s \cdot e[k]$.
  4. Apply anti-windup clamping if the output is saturated.
  5. Compute the filtered derivative: $D[k] = \alpha\, D[k-1] + (1-\alpha)\,\frac{K_d}{T_s}(e[k]-e[k-1])$.
  6. Sum all three terms: $u[k] = K_p\,e[k] + I[k] + D[k]$.
  7. Clamp the output to actuator limits $[u_{\min}, u_{\max}]$.
  8. Store $e[k]$ as $e_{\text{prev}}$ and apply $u[k]$ to the plant.

import numpy as np

class PIDController:
    """
    Discrete-time PID controller with:
      - Anti-windup (clamping method)
      - First-order derivative filter
      - Output saturation
    """
    def __init__(self, Kp, Ki, Kd,
                 dt, u_min=-np.inf, u_max=np.inf, N=10):
        self.Kp, self.Ki, self.Kd = Kp, Ki, Kd
        self.dt = dt
        self.u_min, self.u_max = u_min, u_max

        # Derivative filter coefficient ГЋВ±
        tau_f = Kd / (Kp * N) if Kp > 0 else 1e-3
        self.alpha = tau_f / (tau_f + dt)

        self._integral   = 0.0
        self._prev_error = 0.0
        self._deriv      = 0.0

    def reset(self):
        """Reset internal state (call before new episode)."""
        self._integral = self._prev_error = self._deriv = 0.0

    def compute(self, setpoint, measurement):
        e  = setpoint - measurement
        de = e - self._prev_error

        # Filtered derivative
        self._deriv = (self.alpha * self._deriv
                       + (1 - self.alpha) * self.Kd * de / self.dt)

        # Proportional + derivative (before anti-windup)
        u_pd = self.Kp * e + self._deriv
        u_sat = np.clip(u_pd + self._integral, self.u_min, self.u_max)

        # Anti-windup: only integrate if not saturated
        if u_pd + self._integral == u_sat:
            self._integral += self.Ki * e * self.dt

        self._prev_error = e
        return u_sat

def ziegler_nichols(Ku, Tu, controller_type="PID"):
    """
    Classic Ziegler-Nichols tuning from ultimate gain Ku
    and ultimate period Tu (obtained via relay experiment).

    Returns (Kp, Ki, Kd) gains.
    """
    rules = {
        "P":  (Ku/2,     0,              0),
        "PI":  (0.45*Ku, 0.54*Ku/Tu,    0),
        "PID": (0.6*Ku,  1.2*Ku/Tu,  0.075*Ku*Tu),
    }
    return rules[controller_type]

def relay_experiment(plant_fn, dt=0.01, d=1.0, t_max=30):
    """
    Relay feedback test: drives plant into sustained oscillation
    to identify Ku and Tu automatically.

    plant_fn: callable(u) -> y  (simulate one step)
    d:        relay amplitude
    Returns:  (Ku, Tu)
    """
    y, u    = 0.0, d
    peaks, peak_times = [], []
    prev_y  = 0.0
    t       = 0.0

    while t < t_max:
        y = plant_fn(u)
        u = d if y < 0 else -d

        # Detect peaks (sign change in derivative)
        if prev_y < y and t > 1:
            # Upward peak ГўВЂВ“ record
            peaks.append(y)
            peak_times.append(t)
        prev_y = y
        t += dt

    Tu = 2 * (peak_times[-1] - peak_times[-2]) if len(peak_times) >= 2 else None
     Ku = 4 * d / (3.14159 * np.mean(peaks[-4:]))
    return Ku, Tu

def plant_step(x, u, dt=0.01):
    """Euler integration of ГЎВєВ‹ = -x + u"""
    return x + dt * (-x + u)

# ГўВ”ВЂГўВ”ВЂ Simulation ГўВ”ВЂГўВ”ВЂ
dt     = 0.01
T      = 10.0
t      = np.arange(0, T, dt)

pid = PIDController(Kp=3.0, Ki=1.5, Kd=0.8,
                     dt=dt, u_min=-5, u_max=5)

y_log, u_log, e_log = [], [], []
y = 0.0

for k in range(len(t)):
    u = pid.compute(r[k], y)
    y = plant_step(y, u, dt)
    y_log.append(y)
    u_log.append(u)
    e_log.append(r[k] - y)
