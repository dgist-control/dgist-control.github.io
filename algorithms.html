How It Works

The PID control law computes a control signal u(t) from the tracking error e(t) = r(t) − y(t):

u(t) = Kp·e(t) + Ki·∫e(τ)dτ + Kd·de(t)/dt
  • Proportional (Kp): Reacts to the current error. Higher values give a faster response but can cause overshoot.
  • Integral (Ki): Eliminates steady-state error by accumulating past errors over time.
  • Derivative (Kd): Predicts future error from its rate of change, providing damping and reducing overshoot.

Linear Quadratic Regulator (LQR)

LQR is an optimal state-feedback control method that minimises a quadratic cost function balancing state deviation and control effort.

Problem Formulation

Given the linear system:

ẋ = Ax + Bu,   y = Cx

LQR finds the feedback gain K such that u = −Kx minimises:

J = ∫₀^∞ (xᵀQx + uᵀRu) dt
  • Q ≥ 0 weights the state deviation penalty.
  • R > 0 weights the control effort penalty.
  • The optimal gain is K = R⁻¹BᵀP, where P solves the Algebraic Riccati Equation (ARE).

Key Properties

  • Guarantees at least 6 dB gain margin and ±60° phase margin (for single-input systems).
  • Closed-loop stability is ensured when the pair (A, B) is stabilisable and (A, Q½) is detectable.
  • Commonly paired with a Kalman filter (LQG) for output-feedback control.

Robust Control (H∞)

H∞ synthesis designs a controller that minimises the worst-case gain from disturbance inputs to regulated outputs, providing robustness against model uncertainty.

H∞ Norm

For a stable transfer function T(s), the H∞ norm is the peak singular value over all frequencies:

‖T‖∞ = sup_ω σ̄[T(jω)]

The H∞ control problem seeks a controller C(s) such that the closed-loop norm satisfies ‖T_zw‖∞ < γ for a given performance level γ.

Advantages over Classical Methods

  • Handles multi-input multi-output (MIMO) plants directly.
  • Provides formal robustness guarantees against structured and unstructured uncertainty.
  • Allows systematic frequency-domain performance shaping via weighting functions.
